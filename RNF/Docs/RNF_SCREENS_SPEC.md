
# RNF Screens Specification

Each screen must follow RNF design system.

---

# Screen 1 – Splash

Purpose:
Display RNF branding.

Hierarchy:

Symbol  
RNF  
Tagline

Tagline:

Turn daily habits into lasting discipline.

Duration:
1.2–1.8 seconds.

Then navigate to:

Onboarding or Home.

---

# Screen 2 – Login

Components:

RNF logo  
Title  
Email field  
Password field  
Continue button

Optional:

Sign in with Apple.

---

# Screen 3 – Daily Rhythm

Purpose:

Capture notification times.

Inputs:

Morning Quote time
Default 07:00

Evening Reflection
Default 20:30

Continue button schedules notifications.

---

# Screen 4 – 90 Day Commitment

Title:

Commit to 90 Days

Text:

This is structured progression.

Checkbox:

I understand this requires daily consistency.

Button:

Begin Day 1

Disabled until checkbox selected.

---

# Screen 5 – Home

Displays:

Day number  
Today’s habits  
Completion checkboxes  
XP rewards  
Progress bar

Button:

View Calendar

Tab bar visible.
