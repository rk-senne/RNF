# RNF Design System

This document defines the visual design system for the RNF application.

The goal is visual consistency and calm discipline.

RNF should feel:

minimal  
focused  
premium  
structured  

Avoid:

loud gradients  
confetti animations  
visual clutter  

The interface should feel like a **personal operating system for discipline**.

---

RNF Quests

---

17pt

---

Caption

13pt

---

# Spacing System

Spacing must follow fixed increments.

Small  
8pt

Medium  
16pt

Large  
24pt

Extra Large  
32pt

All layout spacing must use these constants.

---

# Corner Radius

Cards and UI elements use consistent radius.

Standard

16pt

Used for:

Habit cards  
Containers  

---

# Elevation

RNF uses subtle shadows only.

Shadow

opacity 0.1  
radius 8

Used for cards.

Avoid heavy shadows.

---

# Animation Principles

Animations should feel calm.

Avoid excessive motion.

Use:

easeInOut  
spring animations  

Example animations:

XP progress  
Radar chart updates  
Level aura glow

---

# Habit Card Design

Habit cards are the core UI element.

Layout:

Title  
Description  
Completion indicator

States:

Incomplete  
Completed

Completed cards should appear:

slightly darker background  
green check icon
