# RNF Retention System

This document defines the behavioral design systems used to keep users consistently engaged with RNF.

RNF is not designed as a productivity tool.

RNF is designed as a **behavior shaping system**.

The goal is long-term consistency.

Retention is achieved through:

- identity progression
- visible growth
- structured daily loops
- psychological rewards
- predictable structure

---

# Core Retention Loop

RNF relies on a daily behavioral loop.

User opens RNF
→ sees today's quests
→ completes habits
→ gains XP
→ sees character progress
→ maintains streak
→ unlocks identity markers
→ returns tomorrow

Loop:

Open → Act → Reward → Progress → Identity → Return

---

# Daily Engagement Trigger

Users must have a reason to open the app every day.

RNF uses two triggers.

Morning trigger  
Evening trigger

---

## Morning Trigger

Morning notifications reinforce intentional action.

Example notification:

"Begin the day with discipline."

or

"Small actions forge the future."

Purpose:

Prompt the user to begin their daily loop.

---

## Evening Trigger

Evening reminders reinforce reflection.

Example notification:

"Finish strong. Complete today's habits."

Purpose:

Encourage completion before the day ends.

---

# Visible Progress

Progress must always be visible.

RNF provides three visible progress indicators.

XP progress  
Streak counter  
Calendar history

---

## XP Progress

XP gives short-term reward.

Habit completion produces immediate feedback.

Users should always see:

+XP animation  
XP progress bar movement

This creates immediate reward.

---

## Streak Counter

Streaks reinforce consistency.

Users feel psychological pressure to maintain streaks.

Example display:

🔥 7 Day Streak

Breaking streaks should feel meaningful but not punishing.

Forgiveness tokens soften this effect.

---

## Calendar History

The calendar shows a visual record of discipline.

Each day is represented by a square.

Color meanings:

complete → green  
partial → light green  
missed → grey  
forgiven → green with marker

Users subconsciously want a full calendar.

This drives daily engagement.

---

# Identity Progression

RNF emphasizes identity rather than tasks.

Users unlock titles that represent their growth.

Examples:

Spark Initiate  
Ember Initiate  
Forged Disciple  
Iron Ascendant

Titles appear in AscensionView.

Users feel they are becoming someone.

---

# Adaptive Challenge

RNF dynamically adjusts challenges.

QuestGenerator identifies the player's weakest stat.

Example:

If focus is lowest stat

Recommend:

Read  
Meditate  
Journal

This creates personalized improvement.

---

# Habit Expansion

Habits unlock over time.

Week 1

Drink Water  
No Junk Food

Week 2

Add Movement

Week 3

Add Reading

Gradual habit expansion prevents overwhelm.

---

# Loss Aversion

Loss aversion increases engagement.

Users avoid losing streaks.

RNF mitigates frustration using forgiveness tokens.

Forgiveness tokens:

Earn 1 every 30 days.

Allows recovery from occasional mistakes.

---

# Reward Timing

Rewards should occur at multiple time horizons.

Immediate reward

XP gain animation

Short term reward

Level up

Medium term reward

Titles

Long term reward

90-day transformation

---

# Anticipation

Users should feel anticipation about future progress.

Examples:

Next title unlock  
Next level threshold  
Upcoming habit unlock

These elements motivate continued use.

---

# Calm Gamification

RNF intentionally avoids loud or childish gamification.

Avoid:

confetti  
loud sounds  
aggressive colors

Use:

subtle animations  
calm feedback  
structured progression

The system should feel like **personal mastery**, not a game arcade.

---

# Long-Term Retention Systems (Phase 2)

These systems expand retention once the core loop is validated.

Skill Trees

Allow specialization in character growth.

Perk System

Unlock bonuses for long-term players.

Boss Battles

Users defeat symbolic enemies such as:

Procrastination  
Doubt  
Laziness

Guilds

Users join small accountability groups.

AI Coaching

AI generates quests based on behavior patterns.

---

# Key Retention Metrics

RNF success will be measured using retention metrics.

Day 1 retention  
Day 7 retention  
Day 30 retention

Primary target for Phase 1:

Day 7 retention ≥ 40%

If retention exceeds this threshold, advanced systems can be introduced.

---

# Retention Philosophy

RNF is not about motivation.

RNF is about structure.

Motivation fades.

Structure remains.

The app should always reinforce the feeling:

"I am building discipline."
